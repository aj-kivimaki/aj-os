# AJ-OS

AJ-OS is a code-first Notion operating system for managing a freelance game audio business.

The project generates and synchronizes a complete Notion workspace using the Notion API.

## Vision

AJ-OS is designed to manage every aspect of a freelance game audio business, including:

- Projects
- Clients
- Portfolio
- Game Jams
- Production Music
- Learning
- Finance
- Weekly Reviews

## Philosophy

- Code First
- Documentation Driven
- Strongly Typed
- Modular
- Idempotent
