# Development Workflow

Every milestone follows the same lifecycle.

1. Architecture
2. Documentation
3. AI Task
4. Implementation
5. Review
6. Testing
7. Commit
8. Release

Every milestone should leave AJ-OS in a working state.

Documentation always precedes implementation.
