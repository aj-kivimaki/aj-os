# ADR-001: Code First Development

## Status

Accepted

---

## Context

AJ-OS manages the structure of a Notion workspace.

Manually creating or modifying databases inside Notion would make the workspace diverge from the codebase.

---

## Decision

AJ-OS follows a code-first architecture.

The TypeScript application is the single source of truth.

Database schemas, relations, templates and dashboards are generated by code.

---

## Consequences

Benefits

- Version control
- Repeatable deployments
- Easier maintenance
- Reproducible workspaces
- Automated upgrades

Trade-offs

- Initial development requires more planning.
- Manual experimentation inside Notion is discouraged.
