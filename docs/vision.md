# AJ-OS Vision

## Purpose

AJ-OS is the operating system for my freelance game audio business.

Rather than acting as a simple task manager, AJ-OS manages every aspect of my business:

- Projects
- Clients
- Networking
- Portfolio
- Production music
- Learning
- Finances
- Weekly planning
- Long-term goals

Everything should be generated through code using the Notion API.

---

## Mission

Become a respected freelance game audio professional specializing in:

- Music Composition
- Sound Design
- Unity
- Wwise
- Interactive Audio

while building multiple income streams.

---

## Income Streams

Primary

- Freelance Game Audio

Secondary

- Live Performance

Long-Term

- Production Music

---

## Core Principles

AJ-OS should:

- Reduce mental overhead.
- Automate repetitive work.
- Show today's priorities immediately.
- Keep long-term goals visible.
- Connect related information.
- Scale with my business.

---

## Long-Term Goal

AJ-OS should become the single place I open every morning to run my business.
