# AJ-OS Vision

## Purpose

AJ-OS is the operating system for my freelance game audio business.

Rather than acting as a simple task manager, AJ-OS manages every aspect of my business:

- Projects
- Clients
- Networking
- Portfolio
- Production Music
- Learning
- Finances
- Weekly Planning
- Goals
- Knowledge

Everything is generated and maintained through code using the Notion API.

The TypeScript application is the single source of truth.

---

# Mission

Build a sustainable freelance game audio business centered around:

- Music Composition
- Sound Design
- Unity
- Wwise
- Interactive Audio

while continuously improving my technical skills, expanding my professional network and building long-term recurring income.
