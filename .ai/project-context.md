# AJ-OS Project Context

AJ-OS is a code-first Notion Operating System for managing a freelance game audio business.

The TypeScript application is the single source of truth.

The Notion workspace is generated and maintained through the Notion API.

This project is intended to be production quality software.

---

## Purpose

AJ-OS manages:

- Projects
- Clients
- CRM
- Portfolio
- Production Music
- Game Jams
- Learning
- Finance
- Reviews
- Assets
