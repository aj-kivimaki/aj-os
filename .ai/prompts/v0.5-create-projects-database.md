# Objective

Use the existing architecture to create the first real Notion database.

The complete flow should be:

Projects Module

↓

Schema Engine

↓

Notion Translator

↓

Notion Client

↓

Notion Database

No synchronization.

No updates.

No deletion.

Only database creation.

---

# Requirements

Create:

src/sync/

Suggested structure:

```
src/sync/

create-database.ts
index.ts
```

Implement a function that:

1. Accepts a DatabaseDefinition.
2. Uses the existing NotionTranslator.
3. Uses the existing Notion client.
4. Creates a database inside the configured parent page.
5. Returns useful information about the created database.

Add a small executable example (or CLI entry point) that creates the Projects database.

The implementation should reuse all existing layers.

No duplicate translation logic.
