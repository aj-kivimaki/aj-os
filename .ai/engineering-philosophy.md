# AJ-OS Engineering Philosophy

AJ-OS is built like a professional software product.

The goal is not simply to automate Notion.

The goal is to create a maintainable operating system that supports a freelance game audio business for many years.

---

# Core Values

Every decision should favor:

- Simplicity
- Maintainability
- Documentation
- Strong Typing
- Modularity
- Long-Term Scalability

---

# Source of Truth

The TypeScript application defines the system.

Notion is the presentation layer.

Architecture is documented before implementation.

---

# AI Collaboration

ChatGPT designs the architecture.

Cursor implements the architecture.

Architectural decisions belong to humans.

AI assists implementation.

---

# Development Philosophy

Small milestones.

Small commits.

Working software after every milestone.

Documentation before implementation.

Review before merge.

---

# Long-Term Goal

AJ-OS should become a production-quality application that could be open sourced without requiring major architectural changes.

The repository should tell the complete story of how AJ-OS was designed, built and evolved.
