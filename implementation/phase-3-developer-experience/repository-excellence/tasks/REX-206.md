# Purpose

**Why does this task exist?**

`package.json` has **eight top-level keys**. It is missing `description`, `repository`, `author`,
`keywords`, `engines`, `packageManager` — and **`license`, despite a real MIT `LICENSE` file sitting
at the repository root** (F-032). `private: true` excuses `exports` and `files`. **It does not excuse
`license` or `engines`.**

There is **no Node version pin** at all (F-033) — no `engines`, no `.nvmrc`. The repository runs on
whatever the developer happens to have.

There is **no `SECURITY.md`** (F-035) — in a project that handles `ANTHROPIC_API_KEY` and
`API_AUTH_TOKEN`, and whose `configuration.md` has a **§Security** section telling users to protect
them. **A project that tells users to guard secrets and offers no way to report a leak.** No
`CODE_OF_CONDUCT.md` either, though README:22-24 says AJ-OS is *"developed in the open so others can
adapt it."*

**These were allocated with `.github/` by path in the frozen plan.** They are not governance
enforcement; they are **what the repository says about itself**. The re-read separated them.

Make the repository's self-description complete and true.

# Implementation Notes

**Every field must come from a source, not an invention.** `license` from `LICENSE`; `repository`
from the git remote; `author` from `LICENSE`'s copyright line. **A `package.json` that disagrees with
`LICENSE` is worse than one that omits the field** — it creates a contradiction where there was a
gap.

**`SECURITY.md` must be honest about what this project is.** A personal, single-maintainer repository
should not promise a 24-hour triage SLA. State: how to report, what is in scope (`ANTHROPIC_API_KEY`
handling, the `API_AUTH_TOKEN` bearer auth, path-traversal in the handbook/review stores), and what
is not. **A security policy that overpromises is worse than none — it converts a gap into a broken
commitment.**

**Pin Node to one version, in three places that agree**: `engines`, `.nvmrc`, CI. **Three sources
that can disagree is exactly the hand-maintained-copy defect M1 spent a milestone correcting.**
