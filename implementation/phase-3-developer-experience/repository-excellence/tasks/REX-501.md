# Purpose

Comments that restate code, defend deleted constraints, or duplicate JSDoc are noise that drifts —
and the agent layer has almost none where it needs them. This task makes every comment earn its place
per [REX-D6](../decisions/REX-D6.md), **without deleting the irreplaceable knowledge only a comment can
carry.** The preserve-list is the safety net.
